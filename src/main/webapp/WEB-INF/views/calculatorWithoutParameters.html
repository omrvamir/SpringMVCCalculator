Calculator

Вы не ввели параметры в URL или ввели их неверно

Для использования калькулятора, дополните URL запрос: ?a=123&b=456&action=(любую операцию из списка ниже, без скобок)

Вместо 123 и 456 введите свои цифры
Пример: http://localhost:8080/first/calculator?a=10&b=5&action=addition

multiplication = умножение

addition = сложение

subtraction = вычитание

division = деление
